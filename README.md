# 🚀 Doable
### *The Missing Layer Between AI and Execution*

> Built for **Not Your College (NYC) CodeQuest Hackathon 2026**

## 📌 Problem

AI has made learning easier than ever.

People ask AI how to learn a skill, prepare for a hackathon, crack interviews, or stay consistent—but rarely follow through.

The problem is no longer **access to information**.

The problem is **execution**.

---

## 💡 Our Solution

**Doable = AI + Planner + Accountability**

Instead of only answering your questions, Doable helps you achieve your goal.

- 🧠 Understand your goal
- 🗺️ Generate a personalized roadmap
- ✅ Break it into small actionable tasks
- 📷 Submit proof of completion
- 🌍 Watch your virtual world grow as you make real-world progress

Our goal is simple:

> **Help people finish what they start.**

---

## ✨ Features

- AI-powered goal planning
- Personalized roadmaps
- Step-by-step task execution
- Proof submission workflow
- Interactive 3D world
- Progress dashboard
- Responsive modern UI

---

## 🛠 Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, shadcn/ui
- **3D:** Three.js, React Three Fiber
- **Animations:** Framer Motion
- **Icons:** Lucide React

---

## 🚀 Run Locally

```bash
npm install
npm run dev
```

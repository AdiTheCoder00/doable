# Doable

**Doable** is an AI-powered execution platform for builders. Instead of doing the work for you, Doable breaks your goals into actionable roadmaps that you must complete yourself. 

Prove your progress to earn streaks, unlock themes, and watch your interactive 3D island grow!

## ✨ Features
- **AI Roadmaps:** Turn abstract goals into concrete, step-by-step plans.
- **Proof of Work:** Submit screenshots or notes to unlock the next step—no shortcuts.
- **Interactive 3D World:** Watch your personal island expand with new buildings as you complete tasks.
- **Gamified:** Maintain daily streaks and unlock premium visual themes.
- **Local First:** Everything saves instantly and securely to your browser.

## 🛠 Tech Stack
React 18 • TypeScript • Tailwind CSS • Three.js (R3F) • Framer Motion

## 🚀 Quick Start

```bash
git clone https://github.com/AdiTheCoder00/doable.git
cd doable
npm install
npm run dev
```
Visit `http://localhost:5173` to start doing.

# 🚀 Doable
### *The Missing Layer Between AI and Execution*

> **Built for the Not Your College (NYC) CodeQuest Hackathon 2026**

---

## 🌍 Overview

Every day, millions of people ask AI questions like:
- *How do I learn Web Development?*
- *How do I prepare for a Hackathon?*
- *How do I crack interviews?*
- *How do I stay consistent?*

AI provides excellent answers, yet most users never take the first step. The problem isn't finding information anymore; the problem is **executing it consistently.** That observation led us to build **Doable**.

---

## ❌ The Problem

Modern AI assistants are incredible at generating knowledge. They can explain concepts, generate roadmaps, and answer almost anything. However, they all stop at the same place: **The answer.**

Once the conversation ends, there is no accountability, no structured execution, no progress tracking, and no motivation to continue. Users often read the response, bookmark it, promise to start tomorrow, and completely forget about it. Days later, they ask the exact same question again. 

The gap between **knowing** and **doing** is what inspired Doable.

---

## 💡 Our Solution

Instead of acting like another chatbot, Doable transforms every goal into a guided journey. Whether your goal is to learn React, participate in a hackathon, or build a portfolio, Doable doesn't just explain it. It helps you finish it.

**Doable is AI + Planner + Accountability.**

---

## ⚙️ How Doable Works

1. **Describe your goal**: Simply tell the AI what you want to achieve (e.g., "I want to participate in a Hackathon.")
2. **AI Generates a Roadmap**: The AI understands your objective and generates a clear explanation, a structured roadmap, and small, actionable tasks.
3. **Focus on one task**: Instead of overwhelming you with dozens of steps, Doable unlocks progress one task at a time.
4. **Submit Proof**: After completing a task, you must upload proof of completion (currently demonstrated as an MVP) to unlock the next step.
5. **Build your world**: Instead of meaningless points or badges, every completed task contributes to building a personalized virtual 3D world. It serves as a visual reminder that consistency creates progress!

---

## ✨ What Makes Doable Different?

Most productivity tools answer one of two questions: *What should I do?* or *What have I completed?*

Doable focuses on something different: **How do we help users actually finish?** Our platform combines AI guidance, structured planning, accountability, and progress visualization into one seamless experience.

---

## 🎯 Key Features

- **🤖 AI Workspace**: Generate personalized roadmaps from natural language goals.
- **📍 Smart Planning**: Break large goals into smaller, achievable milestones.
- **📋 Guided Task Flow**: Complete one step at a time without feeling overwhelmed.
- **📷 Proof Submission**: Upload proof after completing tasks to maintain accountability.
- **🌍 Interactive 3D World**: Every completed task contributes to a growing virtual environment.
- **📊 Dashboard**: Track your Current Goal, Progress, Activity, and Growth from one place.
- **🎨 Theme Support**: Multiple visual themes for a personalized experience.
- **⚡ Smooth User Experience**: Modern animations and transitions create an engaging workflow.

## 🚀 Running Locally

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build production & Preview:
```bash
npm run build
npm run preview
```

## 🔮 Future Scope

Although this is an MVP built during the NYC CodeQuest Hackathon, the platform has significant potential for expansion:

- **AI Proof Verification**: Automatically verify screenshots using Computer Vision.
- **Adaptive Roadmaps**: Roadmaps that evolve according to user performance.
- **AI Memory**: Remember previous goals and personalize future learning.
- **Calendar Integration**: Sync tasks with Google Calendar.
- **GitHub Integration**: Verify commits automatically for coding goals.
- **Team Goals**: Build accountability with friends or teammates.
- **Mobile Application**: Android and iOS support.
- **Community Challenges**: Learn together through public challenges.
